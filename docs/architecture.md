# Architecture

## Purpose

A vertical mobile game/scene prototype with custom assets, mobile rendering settings and Godot scene structure.

## Main flow

`	ext
Godot project -> Main scene -> scripts/assets -> mobile renderer -> playable prototype
`

## Design notes

This repo is packaged for portfolio review rather than as a store-ready game release.

## Portfolio note

This repository is packaged for review. Some runtime integrations require local credentials or external services and are represented with .env.example instead of real secrets.
